## 1. Motivation：为什么需要这些方法？

DFT（密度泛函理论）从原则上给出了求解多电子体系基态的精确框架。然而，**理论上完美 ≠ 数值上可行**。将 KS 方程付诸计算，必须解决三个核心问题：

| 问题 | 困难所在 | 本讲给出的解法 |
|---|---|---|
| 芯层电子的处理 | 波函数在核附近振荡剧烈，展开需要极多平面波 | 赝势近似 |
| 基底的选取与截断 | 平面波基组理论上无穷大 | 动能截断 Ecut |
| 布里渊区积分 | k 空间连续积分无法直接数值化 | k 空间取样（MP 网格） |
| 非周期体系的处理 | 分子/表面/缺陷本身不具有平移对称性 | 超级原胞 |

## 2. 求解 Kohn-Sham 方程的方法总览

KS 方程的形式为（原子单位）：

```
[-1/2 ∇² + V_eff(r)] Φ_j(r) = ε_j Φ_j(r)
```

求解这一方程，核心是**选取什么基底来展开波函数**：

```
全电子方法
├── FLAPW（全势线性缀加平面波）
│     优点：非常精确，可计算力
│     缺点：计算代价极高
│     代表代码：WIEN2k, EXCITING
└── FP-LMTO（全势线性Muffin-Tin轨道）
      优点：紧凑结构，处理3d/4d/5d金属均匀
      缺点：力的计算有困难

赝势近似
├── 平面波（Plane Waves）★ 最常用
│     优点：力、声子均可计算；仅一个参数Ecut控制精度；FFT加速
│     缺点：局域电子需要极大基组；需生成并检验赝势
│     代表代码：VASP, ABINIT, Quantum ESPRESSO, CPMD
├── 局域轨道（Localized Orbitals）
│     优点：速度快，内存小
│     缺点：需要构造赝势和基底
│     代表代码：SIESTA, OpenMX
└── 数值原子轨道（Numerical Atomic Orbitals）
      优点：速度极快，内存小
      缺点：同上
      代表代码：SIESTA
```

> **Note:** 本讲聚焦于**赝势 + 平面波**这一路线，这也是 VASP、QE 等主流软件的核心方案。

## 3. 赝势理论 Pseudopotential

### 3.1 为什么需要赝势？

**根本矛盾**：影响材料性质的主要是**价电子**，但全电子计算中芯层电子会使波函数在核附近产生剧烈振荡（节点），导致平面波展开需要极多基函数，计算量暴增。

**朴素想法**：把内壳层电子"藏起来"，用一个等效势代替"原子核 + 芯电子"的组合，只对价电子显式求解。

### 3.2 赝势的概念

**赝势（Pseudopotential）** 是一种针对原子构造的近似势，用于替代全电子势。其核心思想：

- 隐去内壳层电子，只保留价电子自由度
- 价电子的**赝波函数（pseudo wavefunction）在截断半径 r_c 内无节点**，光滑可微
- 在 r > r_c 区域，赝波函数与全电子波函数**完全一致**

**直觉理解**：

```
全电子视角：                赝势视角：
[核 + 芯电子 + 价电子]  →  [(核+芯电子)等效势 + 价电子]

真实势：V(r) → -Z/r（趋于负无穷）
赝势：  V_ps(r) → 有限值（光滑）
```

### 3.3 典型例子

**Si 原子：**

```
全电子构型：1s² 2s² 2p⁶ | 3s² 3p²
                芯层(10e)     价层(4e)

Si 赝原子构型：只保留 3s² 3p²（4个电子）
→ 自洽计算只需处理 4 个电子，效率大幅提升
```

**Au 原子：**

```
全电子构型：[Xe] 4f¹⁴ | 5d¹⁰ 6s¹
                内层        价层

Au 赝原子：5d¹⁰ 6s¹（11个电子）
注意：填满的 5d 壳层也可以作为"价电子"处理！
→ 适当扩大价电子范围，可提高赝势的迁移性
```

### 3.4 构造赝势的基本条件

一个合格的赝势必须满足（以角动量 l 对应的径向分量为例）：

**条件 1：无节点**
赝波函数在截断半径内没有径向节点

**条件 2：截断半径外一致（模守恒条件之一）**

```
R_l^PP(r) = R_l^AE(r)，对 r > r_cl
```

**条件 3：模守恒（Norm-conservation）**

```
∫₀^r_cl |R_l^PP(r)|² r² dr = ∫₀^r_cl |R_l^AE(r)|² r² dr
```

即：截断半径内，赝波函数与全电子波函数给出的电荷量相同

**条件 4：本征值一致**

```
ε_l^PP = ε_l^AE
```

对给定角动量的价电子本征能量，两者必须相等

**条件 5（可选）：对数导数连续**
在 r = r_cl 处，两个波函数的对数导数相等，保证散射性质一致

### 3.5 赝势的一般形式（算符形式）

赝势算符的形式为：

```
V̂^ps = Σ_{lm} |lm⟩ V_l ⟨lm|
```

这是一个**投影算符**，对不同角动量分量 (l,m) 施加不同的局域势 V_l。

**为什么是这种形式？** 推导思路：

1. 在单电子近似下，价态波函数 φ_v 和芯态 φ_c 满足正交性：`⟨φ_c|φ_v⟩ = 0`
2. 构造赝波函数 `|φ_v^ps⟩ = |φ_v⟩ + Σ_c α_c|φ_c⟩`，其中 `α_c = ⟨φ_c|φ_v^ps⟩`
3. 对两侧作用 `(H - ε_v)`，利用 KS 本征方程化简，最终得到：

```
(H - V^ps)|φ_v^ps⟩ = ε_v|φ_v^ps⟩

其中 V^ps = Σ_c (ε_c - ε_v)|φ_c⟩⟨φ_c|
```

> **Note:** 由于 ε_c < ε_v（芯层能量低于价层），V^ps 是正的排斥势，这正是赝势使波函数在核附近变光滑（去除节点）的物理原因。

### 3.6 赝势的不唯一性及截断半径的权衡

```
赝势型式不唯一 → 不同赝势可有不同绝对总能量，但能级（本征值）不变
                  → 绝对总能量无意义，只有能量差有物理意义！
```

**截断半径 r_c 的选择权衡：**

| r_c 大 | r_c 小 |
|---|---|
| 计算量小（需要平面波少） | 计算量大（需要更多平面波） |
| 迁移性差（对不同化学环境适应性弱） | 迁移性好（更精确，可移植） |

### 3.7 赝势的检测

赝势不能直接使用，必须经过验证：

- 计算除生成赝势之外的其他原子态，与全电子对比
- 可迁移性检测：通过对数导数比较
- 计算小分子，与全电子结果比较几何、振动频率、偶极矩
- 对不同 r_c，测试基底收敛性

## 4. 平面波基底与动能截断

### 4.1 平移对称性与 Bloch 定理

晶体具有平移对称性：`H(r + t_n) = H(r)`

由此推导 Bloch 定理：

```
Ψ(r + t_n) = e^{ik·t_n} Ψ(r)

等价地：Ψ(r) = e^{ik·r} u(r)，其中 u(r) = u(r + t_n)（周期函数）
```

**意义**：利用平移对称性后，KS 方程从"考虑所有原子"约化为"只考虑原胞内原子"。

从 `H Φ_j(r) = E_j Φ_j(r)` 变为 `H Φ_j^k(r) = E_j(k) Φ_j^k(r)`

### 4.2 平面波展开

利用 Bloch 定理，Bloch 函数可展开为：

```
Φ_{jk}(r) = Σ_G C_{jk}(k+G) · e^{i(k+G)·r}
```

其中：
- `G`：倒格矢
- `k`：第一布里渊区内的波矢
- `C_{jk}(k+G)`：展开系数（待求）
- `e^{i(k+G)·r}`：平面波基函数

### 4.3 动能截断（Energy Cutoff）

**问题**：G 的数目理论上无穷多（平面波基组无穷维），无法直接数值求解。

**解决方案**：用截断能 E_cut 限制平面波数目，只保留动能小于截断值的平面波：

```
ħ²/(2m) · |k + G|² ≤ E_cutoff
```

**关键要点：**

1. **无穷维来源**：平面波基组由所有倒格矢 G 构成，理论上无穷多个
2. **截断必要性**：无法数值求解无穷维矩阵，必须截断到有限维
3. **物理依据**：高能（高频）分量对低能基态贡献可忽略不计
4. **赝势配合**：赝势使波函数在核附近光滑 → 降低所需 E_cut → 计算更高效
5. **收敛性**：增大 E_cut，总能量单调趋于收敛（可系统改善精度）

> **Note:** 动能截断导致基底不完备，这是平面波方法的系统误差来源之一。但它有一个优点：误差可以通过增大 E_cut 系统地减小。

**收敛测试示例（Si 8 原子）：**

```
E_cut 较小时，总能量偏高且不稳定
随 E_cut 增大，总能量单调下降并收敛
Si 的合理截断：约 13 Hartree = 26 Ry ≈ 350 eV
```

图示建议：绘制 Total Energy (eV) vs. Cut-off Energy (eV) 曲线，观察收敛平台。

## 5. K 空间取样

### 5.1 为什么需要 k 空间取样？

布里渊区中的物理量（如电子密度）需要对 k 空间积分：

```
ρ(r) = Σ_k |Φ_k(r)|²
```

这一积分无法连续计算，必须离散化（**k 采样**）。

**核心思想**：相邻 k 点的波函数几乎相同，可以用一个代表性 k 点代表一片区域，并赋予相应权重：

```
ρ(r) = Σ_{k_α ∈ Monkhorst} w_α |Φ_{k_α}(r)|²
```

- `k_α`：Monkhorst-Pack 网格上的采样点
- `w_α`：该 k 点经对称性约化后的权重

### 5.2 Monkhorst-Pack（MP）方法

**方案**：在第一布里渊区中均匀分布 k 点：

```
k_{lmn} = t_l B̄₁ + t_m B̄₂ + t_n B̄₃

t_l = (2l - N₁ - 1) / (2N₁)，  l = 1, 2, ..., N₁
t_m = (2m - N₂ - 1) / (2N₂)，  m = 1, 2, ..., N₂
t_n = (2n - N₃ - 1) / (2N₃)，  n = 1, 2, ..., N₃
```

**MP 网格的关键特性**：网格中心偏离原点（Γ 点），避免 k 点落在高对称点上（除非 N_i 为奇数时自动包含 Γ 点）。

**具体例子：**

```
N₁ = N₂ = 3（正方格子，2D 示意）：
- 共 9 个 k 点
- 包含 Γ 点（N=奇数）
- 经对称性约化后：3 个不可约 k 点
- 权重：α₁=1/9, α₂=4/9, α₃=4/9

N₁ = N₂ = 4：
- 共 16 个 k 点
- 不包含 Γ 点
- 经对称性约化后：3 个不可约 k 点
- 权重：α₁=1/4, α₂=1/2, α₃=1/4
```

### 5.3 实践要点

- **金属体系**：费米面附近 k 点密度不足会引起巨大误差，需要更密的 k 网格（常配合展宽方法 smearing）
- **绝缘/半导体**：相对较稀的 k 网格即可收敛
- **对称性利用**：利用晶体对称性，将 k 点数目约化到不可约布里渊区（IBZ），大幅降低计算量
- **必须收敛测试**：逐步增大 N₁ × N₂ × N₃，直到总能量和目标物理量收敛

## 6. 超级原胞 Supercell

### 6.1 动机

平面波方法本质上依赖**周期性边界条件（PBC）**。但很多实际体系不具备周期性：

- 孤立分子
- 表面/界面
- 点缺陷（空位、杂质）
- 团簇

**超级原胞方法**：构造一个足够大的重复单元，将目标结构放置其中，使相邻周期性像之间的相互作用可以忽略，从而在形式上保留 PBC。

### 6.2 不同体系的超胞构建

**孤立分子：**

```
策略：在分子周围加足够大的真空区（vacuum）
原则：相邻周期性像之间的波函数几乎不重叠
典型真空层厚度：≥ 10-15 Å
```

**表面/界面：**

```
策略：
1. 构建有限厚度的原子层板（slab model）
2. 在垂直表面方向加真空层
3. 原则：相邻板层之间波函数不重叠，且板层足够厚代表体相

真空层选取原则：相邻原子间波函数几乎没有重叠
```

**点缺陷（空位/杂质）：**

```
策略：取 n×n×n 的超胞（primitive cell 的整数倍），
在其中引入一个缺陷
关键：超胞要足够大，使得缺陷与其周期性像之间的相互作用可忽略
→ 需要做超胞尺寸收敛测试
```

### 6.3 超胞收敛测试

以 NH₃ 分子为例（测试 N-H 键长对超胞尺寸的收敛性）：

```
超胞边长 ≈ 2 Å 时：键长严重偏小
超胞边长 ≈ 4-5 Å 时：接近收敛
超胞边长 ≈ 6-7 Å 时：完全收敛（约 15 a.u.）
```

图示建议：绘制物理量（键长、总能量）vs. 超胞尺寸曲线，确认收敛平台。

## 7. 原子受力计算：Hellmann-Feynman 定理与 Pulay 力

### 7.1 为什么需要计算力？

结构优化（找稳定构型）和分子动力学（AIMD）都需要知道每个原子受到的力：

```
F_μ = -dE_tot / dR_μ
```

其中 `R_μ` 是第 μ 个原子核的坐标，`E_tot` 是总能量（核-核排斥 + 电子体系能量）。

### 7.2 Hellmann-Feynman 定理

**定理内容**：对于精确的（完全自洽的）波函数，一阶波函数变分对力没有贡献，即：

```
F_μ = -dE_tot/dR_μ = -∂E_tot/∂R_μ
```

即**只需对显含原子核坐标的部分求微分**（不需要对隐含在波函数中的部分再次求导）。

数学上，对任意参数 λ，有：

```
⟨ψ| H |∂ψ/∂λ⟩ = E ⟨ψ| ∂ψ/∂λ⟩
```

### 7.3 HF 力的表达式

总力可以分解为：

```
F_μ = F_μ^HF + F_μ^el

F_μ = -∂E_nn/∂R_μ  -  ∫ dr ρ(r) ∂V_ext(r)/∂R_μ  -  ∫ dr (δE_tot/δρ(r)) ∂ρ(r)/∂R_μ
       核-核排斥力项        HF 力项（主项）                     变分力项（Pulay 力）
```

其中：
- `E_nn` 和 `V_ext` 显含 `{R_μ}`（直接依赖核坐标）
- `ρ(r)` 隐含 `{R_μ}`（通过自洽计算间接依赖）

### 7.4 Pulay 力与平面波基底的优势

对于第三项（变分力 / Pulay 力），根据 HF 定理它在精确波函数下应为零。

**关键问题**：这取决于所用基底是否随核坐标移动。

```
平面波基底：e^{i(k+G)·r}
→ 平面波与核坐标无关！
→ ∂ρ(r)/∂R_μ ≡ 0
→ Pulay 力 = 0
→ 力的计算非常简洁，只需计算 HF 力项
```

这是平面波基底的重要优势之一。局域轨道基底（随原子移动）则存在 Pulay 力修正，计算更复杂。

## 8. 总能量最小化算法

### 8.1 问题形式

使用赝势后，KS 方程形式不变：

```
[-1/2 ∇² + V̂_eff(r)] Φ_j(r) = ε_j Φ_j(r)

V̂_eff(r) = Σ_{μ,l} V_l(|r - R_μ|) P_l + V_coul(r) + V̂_xc(r)
```

总能量（形式不变）：

```
E_tot = T + E_ext + E_coul + E_xc(r) + E_nn
```

> **Note:** 由于赝势的不唯一性，绝对总能量无物理意义；只有**能量差**（不同结构之间、不同相之间）才有意义！

**DFT 自洽计算的整体流程：**

```
给定初始核坐标 {R_μ}
    ↓
自洽求解电子密度 ρ(r) → 得到电子体系总能量
    ↓
计算总能量（包含 E_nn）
    ↓
微调核坐标（力沿负梯度方向移动原子）
    ↓
总能量是否最小？
    否 → 回到自洽求解
    是 → 输出收敛结构的性质（能量、力、电子密度、能带等）
```

### 8.2 最速下降法（Steepest Descent）

**思想**：沿负梯度方向（最快下降方向）移动波函数系数。

迭代更新：

```
X_{k+1} = X_k - λ_k ∇F(X_k)

其中下降方向 p_k = -∇F(X_k)（负梯度方向）
```

**问题**：相邻两步方向相互正交（zig-zag 锯齿形路径），在狭长谷底型能量面上收敛极慢。

### 8.3 共轭梯度法（Conjugate Gradient）

**改进**：新的搜索方向由**当前负梯度 + 前一步搜索方向**叠加构成：

```
p_{k+1} = -∇F(X_{k+1}) + β_k p_k

β_k = (∇F(X_{k+1})^T ∇F(X_{k+1})) / (∇F(X_k)^T ∇F(X_k))   [Fletcher-Reeves 公式]
```

**优点**：

- 保留前一步的搜索方向信息，避免 zig-zag
- 对 n 维二次型问题，至多 n 步收敛
- 在大多数 DFT 程序中，实际使用**预条件共轭梯度法（Preconditioned CG）**，收敛更快

图示建议：对比 Steepest Descent 与 CG 在椭圆等高线上的迭代路径。

## 10. 小结与关键公式速查

### 核心近似体系

```
第一性原理计算引入的近似：
1. Born-Oppenheimer（绝热）近似 → 核与电子运动分离
2. 交换关联能近似（LDA/GGA）  → E_xc[ρ] 近似处理
3. 赝势近似                   → 芯电子等效处理
   └─ 包含：核-芯电子相互作用近似
            芯-芯电子相互作用近似
            芯-价电子相互作用近似
```

### 关键公式汇总

```
【Bloch 定理】
Ψ(r + t_n) = e^{ik·t_n} Ψ(r)

【波函数展开（平面波）】
Φ_{jk}(r) = Σ_G C_{jk}(k+G) e^{i(k+G)·r}

【动能截断条件】
ħ²/(2m) |k+G|² ≤ E_cutoff

【电子密度（k采样）】
ρ(r) = Σ_{k_α ∈ MP} w_α |Φ_{k_α}(r)|²

【原子受力（HF定理）】
F_μ = -dE_tot/dR_μ = -∂E_tot/∂R_μ（对平面波基底精确成立）

【赝势算符】
V̂^ps = Σ_{lm} |lm⟩ V_l ⟨lm|

【模守恒条件】
∫₀^{r_cl} |R_l^PP|² r² dr = ∫₀^{r_cl} |R_l^AE|² r² dr
ε_l^PP = ε_l^AE
```

### 计算参数选取原则

| 参数 | 如何确定 | 经验值 |
|---|---|---|
| E_cut | 收敛测试（对总能量或目标物理量） | 氧化物约 400-600 eV，金属约 300-500 eV |
| k 网格 | 收敛测试（对总能量、力） | 绝缘体 4×4×4 起；金属更密 |
| 真空层厚度 | 使相邻像之间相互作用<1 meV | ≥ 15 Å |
| 超胞尺寸（缺陷） | 使缺陷-缺陷相互作用可忽略 | ≥ 3×3×3 primitive cell |
| 截断半径 r_c | 迁移性 vs. 计算效率权衡 | 看具体元素 |

### 常见误区

| 误区 | 正确理解 |
|---|---|
| 绝对总能量可以比较不同赝势结果 | 不能！赝势不同，绝对总能量无可比性，只有能量差有意义 |
| E_cut 越大越好 | 需要平衡精度与计算代价，找到收敛平台即可 |
| k 点越多越好 | 同上，找到收敛点即可，对绝缘体通常不需要极密 k 网格 |
| 超胞稍小也无所谓 | 超胞不够大会引入人为的周期性相互作用，严重影响结果 |
| 全电子方法一定比赝势准确 | 不一定；赝势方法在设计良好时可以达到很高精度 |
