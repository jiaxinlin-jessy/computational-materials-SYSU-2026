## 1. 量子力学基础回顾

### 1.1 五个基本假设（研究生面试高频）

| # | 假设内容 |
|---|---------|
| 1 | 微观体系的运动状态由**归一化波函数** Ψ 描述 |
| 2 | 波函数随时间变化遵从**薛定谔方程** |
| 3 | 可观测量由**厄米算符**（Hermitian operator）表示 |
| 4 | 测量某力学量，结果必为该算符的**本征值之一** |
| 5 | 全同多粒子系的波函数对任意两粒子交换具有**反对称性**（费米子）或对称性（玻色子） |

> **直觉**：假设 5 正是泡利不相容原理的根源——两个费米子若量子数完全相同，交换后波函数要同时满足"等于自身"和"等于负自身"，只有 Ψ = 0，即该态不存在。

## 2. 单电子薛定谔方程

### 2.1 库仑场中的单电子（类氢原子）

**问题简化**：电子-核构成双粒子系统，利用质心坐标分离，由于核质量 M = 1836.1 × 电子质量 m，质心近似在核位置，折合质量 μ ≈ m。

**薛定谔方程**（中心力场）：

```
H_hat Ψ = E Ψ

H_hat = -[h²/(8π²m)] ∇² - Ze²/(4πε₀r)

其中：
  ∇²（拉普拉斯算符）= ∂²/∂x² + ∂²/∂y² + ∂²/∂z²
  在球坐标中展开后含 r, θ, φ 三个变量
```

### 2.2 分离变量

令 `Ψ(r, θ, φ) = R(r) · Θ(θ) · Φ(φ)`，可将方程拆成三个独立常微分方程：

| 方程 | 变量 | 量子数 |
|------|------|--------|
| 径向方程 R(r) | r | 主量子数 n，角量子数 l |
| 极角方程 Θ(θ) | θ | l, m |
| 方位角方程 Φ(φ) | φ | 磁量子数 m |

**方位角方程解**：`Φ(φ) = (1/√2π) · e^(imφ)`

单值条件要求 `e^(i·2πm) = 1`，所以 **m = 0, ±1, ±2, …**

**极角方程**：要求解在 θ = 0 和 θ = π 处不发散，强制 **l 为非负整数**，且 |m| ≤ l。

**径向方程**：幂级数展开后，若不截断则 r→∞ 时指数发散，物理要求截断，截断条件引入**主量子数 n**。

### 2.3 三个量子数的物理意义

```
量子数范围：
  n = 1, 2, 3, ...         （主量子数，决定能量）
  l = 0, 1, 2, ..., n-1   （角量子数，决定角动量大小）
  m = 0, ±1, ±2, ..., ±l  （磁量子数，决定角动量 z 分量）
```

> **直觉**：三维方程 → 三个分离变量 → 三个量子数。每个量子数对应一个物理约束（边界条件）。

#### 对应轨道符号

| l 值 | 0 | 1 | 2 | 3 | 4 | 5 |
|------|---|---|---|---|---|---|
| 符号 | s | p | d | f | g | h |

### 2.4 氢原子能量与简并度

**能量公式**：

```
E_n = -（1/2）·（e²/4πε₀a₀）·（Z/n）²
    = -13.6 × (Z/n)² eV
```

**简并度**：

```
g = Σ(l=0 到 n-1) (2l+1) = n²
```

**节点规律**：波函数 ψ_{n,l,m} 共有 n-1 个节点，其中角向节点 l 个，径向节点 n-l-1 个。

**氢原子光谱（里德堡公式）**：

```
ν = ΔE/hc = R_H · (1/n₁² - 1/n₂²)

里德堡常数 R_H = (1/2) · e²/(4πε₀·a₀·h·c)
```

### 2.5 轨道角动量与自旋

**轨道角动量**：

```
|M| = √[l(l+1)] · ħ,   l = 0, 1, 2, ..., n-1
M_z = m·ħ,             m = 0, ±1, ..., ±l
```

**自旋角动量**（类比轨道）：

```
|M_s| = √[s(s+1)] · ħ,  s = 1/2（固定）
M_{z,s} = m_s · ħ,      m_s = +1/2 或 -1/2
```

> **直觉**：自旋在经典力学无对应量，无法从经典哈密顿量构建。斯特恩-盖拉赫实验（银原子束通过非均匀磁场分成两束）是自旋量子化的直接证据。

**完整量子数**：描述一个电子的状态需要四个量子数 **(n, l, m_l, m_s)**。

## 3. 多电子体系薛定谔方程

### 3.1 多电子哈密顿量（原子单位制）

```
H_hat = Σ_i [-（1/2）∇_i² - Z/r_i]  +  (1/2) Σ_i Σ_{j≠i} 1/r_{ij}

        \_______动能+核吸引_______/     \___电子间排斥（关键！）___/
```

> **难点所在**：电子-电子排斥项是双电子算符（含 r_{ij}），导致方程无法精确分离变量。这就是多电子问题复杂性的根本来源。

**为什么用原子单位（Atomic Units, a.u.）？**
- 使哈密顿量表达式更简洁（消去 e, ħ, m_e 等常数）
- 编程时避免极小量（如 ħ = 1.055×10⁻³⁴ J·s）出现在分母导致数值溢出

**常用原子单位对应 SI**：

| 物理量 | 符号 | SI 数值 |
|--------|------|---------|
| 长度（Bohr 半径）| a₀ | 5.292 × 10⁻¹¹ m |
| 能量（Hartree） | E_H | 4.360 × 10⁻¹⁸ J |
| 质量 | m_e | 9.109 × 10⁻³¹ kg |
| 电荷 | e | 1.602 × 10⁻¹⁹ C |

### 3.2 为什么不能忽略电子-电子相互作用？

**反例（He 原子）**：

```
若忽略 e-e 排斥：
  理论预测 E₁ = -54.4 eV
  实验测量 E₁ = -24.6 eV
```

误差高达 120%，完全不可接受。

## 4. 绝热近似（Born-Oppenheimer）

### 4.1 核心思想

电子质量远小于核质量（M ≈ 1836 m_e），因此**电子运动比核运动快得多**。

> **直觉类比**：就像蜜蜂绕着慢速移动的熊飞翔——蜜蜂（电子）在每个时刻都"看见"一个静止的熊（核），并立刻调整到对应的量子态。

**近似结果**：可以把核坐标 R 当作参数固定，只求解电子薛定谔方程：

```
H_e · Ψ_e(r; R) = E(R) · Ψ_e(r; R)

H_e 中不含核动能项，核-核排斥作为常数加入
```

这样，分子总能量曲面（势能面 PES）是 R 的函数，用于描述化学键、结构优化等。

## 5. Hartree 近似与自洽场方法

### 5.1 单电子近似（平均场思想）

**基本思路**：用其他所有电子产生的**平均静电势**来代替电子-电子的精确排斥，把双电子算符变成单电子算符：

```
H_i = -（1/2）∇_i² - Z/r_i + U(r_i)

U(r_i) = Σ_{j≠i} ∫ |ψ_j(r_j)|² · (1/r_{ij}) · dτ_j
          \_______第 j 个电子电荷密度产生的势_______/
```

**多电子波函数的乘积假设（Hartree 近似）**：

```
Ψ(1, 2, ..., k) = ψ₁(1) · ψ₂(2) · ... · ψ_k(k)
```

由此得到 **Hartree 方程**（一组耦合的单电子方程）：

```
[-（1/2）∇_i² - Z/r_i + U(r_i)] · ψ_i = E_i · ψ_i

总能量修正：E = Σ E_i - (1/2) Σ_{ij} J_{ij}
（减去双计的电子对排斥能）
```

### 5.2 自洽场方法（SCF）

由于 U(r_i) 本身依赖于所有 ψ_j，方程必须**迭代求解**：

```
流程：
1. 猜初始波函数 {ψ_i⁰(x)}
2. 计算有效势 V_eff(r)
3. 求解薛定谔方程 (T_i + V_eff) ψ_i = ε_i ψ_i
4. 检验收敛性 |ψ_i - ψ_i⁰| < 10⁻⁶
5. 若未收敛：令 ψ_i⁰ = ψ_i，返回步骤 2
6. 若收敛：输出结果
```

### 5.3 中心场近似与屏蔽模型

**中心场近似**：进一步假设平均势只与径向距离有关（与角度无关）：

```
U(r_i⃗) ≈ U(r_i)  （去掉方向依赖）
```

这样角向方程与氢原子**完全一致**，只需重新求解径向方程。

**有效核电荷（Slater 规则）**：

```
Z_eff = Z - σ

σ = 其余电子对该电子的屏蔽常数（半经验值）
```

> **直觉**：内层电子"屏蔽"了核电荷，外层电子感受到的是被部分屏蔽的有效核电荷 Z_eff < Z，轨道大小由 Z_eff 决定。

## 6. Hartree-Fock 近似

### 6.1 反对称化：Slater 行列式

Hartree 近似的乘积波函数**不满足费米子反对称性**。修正方法是用 Slater 行列式代替简单乘积：

```
Ψ(1,2,...,k) = (1/√k!) × 行列式

行列式元素：φ_i(j) · η_i(j)   （空间轨道 × 自旋函数）

性质：
  - 交换任意两行 → 行列式变号 ✓（反对称）
  - 两行相同（两电子量子数完全相同）→ 行列式 = 0 ✓（泡利不相容）
```

> **直觉**：Slater 行列式自动实现了"没有两个电子可以占据完全相同量子态"的约束，是泡利原理的数学体现。

### 6.2 HF 方程与交换作用

在 Slater 行列式下计算 ⟨H⟩，比 Hartree 多出**交换积分 K**：

```
Hartree（Coulomb）积分：
  J_{ij} = ∫∫ |ψ_i(1)|² · (1/r₁₂) · |ψ_j(2)|² dτ₁dτ₂

交换积分（Fock 项，无经典对应）：
  K_{ij} = ∫∫ ψ_i*(1)ψ_j*(2) · (1/r₁₂) · ψ_i(2)ψ_j(1) dτ₁dτ₂
```

**自旋关联**：考虑自旋后：

```
平行自旋（↑↑）：U = J - K   （能量较低）
反平行自旋（↑↓）：U = J    （能量较高）
```

这正是**洪特规则（Hund's Rule）**的量子力学来源：平行自旋状态因交换积分 K > 0 而能量更低。

### 6.3 洪特规则（Hund's Rule）

```
规则：同一轨道组中，电子优先以平行自旋单独占据不同轨道，
      而非两个电子同占一个轨道。

量子力学解释：
  平行自旋 → 空间波函数反对称 → 两电子避免近距离出现
  → 减小排斥能 → 能量更低
```

## 7. 价键理论（VB）

### 7.1 H₂ 分子的 BO 近似哈密顿

```
H_hat(r; R) = -（1/2）∇₁² - (1/2)∇₂²
              - 1/r_{a1} - 1/r_{a2} - 1/r_{b1} - 1/r_{b2}
              + 1/R_{ab} + 1/r_{12}
```

其中下标 a,b 为核，1,2 为电子。

### 7.2 Heitler-London 方法（VB 雏形）

**试探波函数**（态叠加，考虑两种等价情形）：

```
Ψ = C₁ψ₁ + C₂ψ₂

ψ₁ = ψ_a(1)ψ_b(2)  （电子1在a核，电子2在b核）
ψ₂ = ψ_a(2)ψ_b(1)  （电子1在b核，电子2在a核）
```

**线性变分法**：最小化 E = ⟨Ψ|H|Ψ⟩/⟨Ψ|Ψ⟩，对 C₁, C₂ 求极值，得到**久期行列式**：

```
| H₁₁-E    H₁₂-ES₁₂ |
| H₁₂-ES₁₂  H₂₂-E   | = 0
```

**解**：

```
E_I  = (H₁₁ + H₁₂) / (1 + S₁₂)     → 成键态（低能，稳定）
E_II = (H₁₁ - H₁₂) / (1 - S₁₂)     → 反键态（高能，不稳定）

对应波函数：
  Ψ_I  = (ψ₁ + ψ₂) / √(2 + 2S₁₂)   → 对称（成键）
  Ψ_II = (ψ₁ - ψ₂) / √(2 - 2S₁₂)   → 反对称（反键）
```

### 7.3 重要积分定义

```
重叠积分：S₁₂ = ∫ψ₁ψ₂dτ = S_ab² = e^{-2R}(1 + R + R²/3)

库仑积分：H₁₁ = E_H + E_H + Q
  Q = (1 + 1/R) · e^{-2R}   （库仑吸引修正）

交换积分：H₁₂ = 2E_H · S_ab + A
  A = (1/R - 2R/3) · e^{-R}  （量子力学特有，驱动成键）
```

> **直觉**：**交换积分 A 才是共价键的本质**——它来源于量子力学的全同粒子不可分辨性，在经典力学中无对应量。成键态的低能量主要由 A < 0 贡献。

### 7.4 计算结果对比

| 方法 | 键长 r₀ | 离解能 D_e |
|------|---------|-----------|
| 实验值 | 0.742 Å | 4.74 eV |
| Heitler-London | 0.87 Å | 3.22 eV |
| 王守竟（改进） | 0.74 Å | 3.76 eV |
| Kolos-Roothaan（精确） | 0.742 Å | 4.74 eV |

### 7.5 价键理论的共价键本质

```
电子密度分析：

成键轨道（Ψ_I）：
  ρ_I = (1/(1+S²)) · [φ_a² + φ_b² + 2φ_a·φ_b·S_ab]
  ρ_I - ρ_经典 = 2φ_a²·S_ab/(1+S²) · (1 - S_ab) > 0
  → 两核间电子密度增加（"建设性干涉"）→ 吸引核 → 成键

反键轨道（Ψ_II）：
  ρ_II - ρ_经典 < 0
  → 两核间电子密度减少（"破坏性干涉"）→ 不成键
```

**共价键两大要素**：
1. 两电子自旋**反平行**（满足泡利原理，允许共享轨道）
2. **最大重叠原理**：键的强度正比于原子轨道的重叠程度

## 8. 分子轨道理论（MO/LCAO）

### 8.1 分子轨道的构造（LCAO 方法）

**思路**：分子中的每个电子在所有原子的平均场中运动，分子轨道由原子轨道线性组合构成：

```
ψ_i = Σ_j C_{ij} · φ_j

（LCAO: Linear Combination of Atomic Orbitals）
```

**关键人物**：Mulliken（MO 理论奠基）、Lennard-Jones、Herzberg、Hund

### 8.2 以 H₂⁺ 为例（单电子双中心）

**方程与试探函数**：

```
H_hat = -(1/2)∇² - 1/r_a - 1/r_b + 1/R_{ab}

ψ = C_a·φ_a + C_b·φ_b
```

**久期行列式**（同 VB 形式）：

```
| H_aa - E       H_ab - E·S_ab |
| H_ab - E·S_ab  H_bb - E      | = 0
```

**解**（利用 H_aa = H_bb，令 α = H_aa，β = H_ab）：

```
E_I  = (α + β) / (1 + S) ≈ α + β    （成键分子轨道，MO）
E_II = (α - β) / (1 - S) ≈ α - β   （反键分子轨道，MO*）

ψ_I  = (φ_a + φ_b) / √(2 + 2S)     （σ 成键）
ψ_II = (φ_a - φ_b) / √(2 - 2S)     （σ* 反键）
```

### 8.3 LCAO 适用条件（原子轨道选择）

构建分子轨道的原子轨道需满足：

1. **对称性匹配**：两个原子轨道的对称性（对称/反对称于分子轴）必须兼容
2. **能量匹配**：α_a ≈ α_b（库仑积分接近），能量差越大，混合越少
3. **空间重叠**：重叠积分 S_ab 不为零（有实际重叠区域）

> **直觉**：能量匹配是"共振条件"——两个能量相近的原子轨道才能有效混合形成分子轨道，就像两个频率相近的振子才能发生共振。

### 8.4 成键与反键的电子密度分析

```
成键（Ψ_I）：ρ = |ψ_I|² ∝ φ_a² + φ_b² + 2φ_a·φ_b
  → 核间区域电子密度增加（建设性干涉）→ 屏蔽核排斥 → 稳定

反键（Ψ_II）：ρ = |ψ_II|² ∝ φ_a² + φ_b² - 2φ_a·φ_b
  → 核间区域有节面，电子密度减少（破坏性干涉）→ 不稳定
```

### 8.5 一般分子（m 核 n 电子）

**分子哈密顿量**：

```
H_hat = Σ_i (-1/2)∇_i²          （电子动能）
       + Σ_a Σ_i (-Z_a/r_{ai})   （核-电子吸引）
       + (1/2)Σ_i Σ_j 1/r_{ij}   （电子-电子排斥）
       + (1/2)Σ_a Σ_b Z_a·Z_b/R_{ab} （核-核排斥）
```

**单电子近似后的总能量**：

```
E = Σ_i E_i + (1/2)Σ_a Σ_b Z_a·Z_b/R_{ab} - (1/2)Σ_i Σ_j J_{ij}
```

## 9. VB 与 MO 方法的对比

| 维度 | 分子轨道理论（MO） | 价键理论（VB） |
|------|------------------|--------------|
| 哈密顿量 | 相同（BO 近似后） | 相同（BO 近似后） |
| 单电子近似 | 使用（单电子 Hamiltonian） | 不使用（直接处理双电子） |
| 试探波函数 | `ψ(1) = C_a·φ_a(1) + C_b·φ_b(1)` | `Ψ(1,2) = C₁·φ_a(1)·φ_b(2) + C₂·φ_b(1)·φ_a(2)` |
| 成键条件 | 不要求完美配对 | 要求两电子自旋完美配对 |
| 成键位置 | **离域成键** | **定域成键** |
| 离子项 | 含有（C₁²·φ_a(1)φ_a(2) + ...） | 无离子项 |
| 适用范围 | 基态 + 激发态 | 主要是基态 |

> **结论**：两种方法是同一问题的不同视角，互相补充。改进的 VB（引入离子项）和改进的 MO（配置相互作用 CI）最终会收敛到相同的精确结果。

## 10. 关联问题与 HF 的局限

### 10.1 关联能的定义

```
E_c = <H> - <H>_{HF} ≤ 0

（精确能量 − HF 能量，由于变分原理，E_c ≤ 0）
```

**数值例子**：

| 体系 | 精确 <H> | HF <H> | E_c |
|------|---------|--------|-----|
| He 原子 | -2.904 | -2.862 | -0.042 |
| Be 原子 | -14.667 | -14.571 | -0.096 |
| Ne 原子 | -128.938 | -128.555 | -0.383 |

### 10.2 关联来源

**动态相关**：HF 将 e-e 排斥平均化，实际上电子之间的瞬时关联（一个电子靠近时另一个会主动回避）被忽略，导致排斥能估计偏高。

**静态相关**：单 Slater 行列式本质上只能描述单一组态，无法正确描述原子间距很大时的解离极限（键断裂时需要多行列式描述）。

### 10.3 HF 的三大问题

1. **计算量随电子数指数增长**：O(N⁴) 或更高，对大体系不可行
2. **无法描述化学键形成与断裂**（缺乏关联，化学精度差）
3. **金属费米能和半导体能带偏差大**

> **引出 DFT**：这三个问题促使人们寻求更好的方法，最终导向密度泛函理论（DFT）——用电子密度 n(r) 代替多体波函数作为基本变量，既降低了计算量，又能部分包含关联效应。

## 11. 核心概念速查表

### 量子数一览

| 量子数 | 符号 | 取值 | 物理意义 |
|--------|------|------|----------|
| 主量子数 | n | 1, 2, 3, ... | 能量、轨道大小 |
| 角量子数 | l | 0, 1, ..., n-1 | 轨道角动量大小 |
| 磁量子数 | m_l | 0, ±1, ..., ±l | 角动量 z 分量 |
| 自旋量子数 | m_s | +1/2, -1/2 | 自旋 z 分量 |

### 关键近似层级

```
精确多体薛定谔方程
    ↓ Born-Oppenheimer（绝热近似）
电子薛定谔方程（固定核）
    ↓ 单电子近似（平均场）
Hartree 方程（乘积波函数）
    ↓ 引入反对称化（Slater 行列式）
Hartree-Fock 方程（HF）
    ↓ 超越 HF（加入关联）
CI / MP2 / CCSD / DFT ...
```

### 重要积分

| 名称 | 符号 | 含义 |
|------|------|------|
| 重叠积分 | S_ab | 两原子轨道的空间重叠程度 |
| 库仑积分 | α（或 J） | 电子留在原子轨道的能量 |
| 共振积分 | β（或 H_ab） | 电子在两中心间跳跃的矩阵元 |
| 交换积分 | K | 量子力学特有，与自旋关联相关 |

### 成键条件速记

```
构成有效化学键，原子轨道需满足：
  1. 对称性匹配（symmetry match）
  2. 能量接近（energy matching，|α_a - α_b| 小）
  3. 空间重叠非零（S_ab ≠ 0）
```
