## 1. Motivation：为什么需要密度泛函？

### 多体问题的困境

量子力学告诉我们：多电子体系的性质完全由多体波函数 `Ψ(r₁, r₂, ..., rN)` 决定。

**问题在于**：

- 波函数包含 **3N 个变量**（N = 电子数）
- 对于一个包含 100 个电子的体系，波函数是一个 300 维的函数
- 直接求解在计算上几乎不可能

**DFT 的核心思想**：

> 将 3N 维的多体波函数问题，转化为只有 3 个空间变量的**电子密度** `ρ(r)` 的问题。

这一转化使得大体系的量子力学计算成为可能。

## 2. Thomas-Fermi 理论：密度泛函的雏形

> **提出者**：L. H. Thomas（1903-1992）& E. Fermi（1901-1954），1927 年  
> **核心观点**：原子的总能量是电子密度的泛函

### 2.1 基本思路

Thomas-Fermi 理论基于 Hartree 总能量出发，将各项用电子密度 `ρ(r)` 表示：

```
⟨H⟩_TF = T_TF[ρ] + ∫ v(r₁)ρ(r₁)dr₁ + (1/2)∬ ρ(r₁)ρ(r₂)/|r₁-r₂| dr₁dr₂
```

三项分别是：**动能** + **外势能（电子-核相互作用）** + **库仑能（电子-电子排斥）**

### 2.2 动能的近似：关键步骤

TF 理论的核心近似是将动能写成密度的局域泛函：

```
T_TF = C_F ∫ ρ^(5/3)(r) dr

其中 C_F = (3/10)(3π²)^(2/3) ≈ 2.871
```

**直觉解释**：把每个空间点处的电子视为局域的均匀自由电子气（费米气体），利用费米球的动能公式积分得到。`ρ^(5/3)` 来源于三维自由电子气的动能密度正比于 `k_F² ∝ ρ^(2/3)`，再乘以 `ρ` 得 `ρ^(5/3)`。

### 2.3 Thomas-Fermi 的贡献与局限

| 方面 | TF 理论 |
|------|---------|
| **贡献** | 首次提出"总能量是电子密度的泛函"这一核心思想 |
| **局限1** | 动能是近似的（局域自由电子气假设太粗糙） |
| **局限2** | 没有交换能、关联能 |
| **局限3** | 对原子、分子描述非常不准确 |

**关键问题**：TF 理论是否只是一个特殊近似，还是说"总能量可以写为 ρ 的泛函"是一个普遍结论？

## 3. Hohenberg-Kohn 定理：DFT 的理论基础

> **提出者**：P. Hohenberg & W. Kohn，1964 年  
> **意义**：严格证明了 DFT 的合法性，不含任何近似

### 3.1 基本设定

在 Born-Oppenheimer（绝热）近似下，给定原子核位置后，电子哈密顿量为：

```
Ĥ_el = T̂_e + Û_ee + V̂_ext

其中：
  T̂_e   = 电子动能
  Û_ee   = 电子-电子相互作用
  V̂_ext  = 外势（来自原子核的库仑势）
```

外势是唯一"随体系变化"的量：

```
v̂_ext(r) = Σ_α (-Z_α) / |R_α - r|
```

### 3.2 定理一（第一 HK 定理）

**表述**：体系基态电子密度 `ρ(r)` 与外势 `V_ext(r)` 之间存在**一一对应关系**（差一个无关紧要的常数）。

> 换言之：**如果你知道了基态电子密度，你原则上就知道了一切。**

- `ρ(r)` 的积分给出电子数 N：`N = ∫ ρ(r) dr`  
- `ρ(r)` 唯一确定 `V_ext(r)`  
- `V_ext(r)` 唯一确定哈密顿量 `Ĥ`  
- `Ĥ` 唯一确定所有性质（波函数、能量、力……）

因此：**所有可观察量都是电子密度 `ρ(r)` 的泛函。**

**证明思路（反证法）**：

假设两个不同的外势 `V₁(r)` 和 `V₂(r)`（差值非常数）对应**同一个**基态密度 `ρ(r)`，设对应基态能量分别为 `E₁`、`E₂`，利用变分原理可得：

```
E₁ < E₂ + ∫ ρ(r)[V₁(r) - V₂(r)] dr
E₂ < E₁ - ∫ ρ(r)[V₁(r) - V₂(r)] dr
```

两式相加得：`E₁ + E₂ < E₁ + E₂`，矛盾！

因此同一个 `ρ(r)` 不可能对应两个不同的外势。**定理得证。**

### 3.3 定理二（第二 HK 定理，变分原理）

**表述**：对任意满足 `ρ'(r) ≥ 0` 且 `∫ ρ'(r) dr = N` 的试探密度，有：

```
E[ρ'(r)] ≥ E₀
```

等号当且仅当 `ρ'(r) = ρ₀(r)`（真实基态密度）时成立。

**意义**：可以通过对密度做变分（能量最小化）来求得基态——**密度可以作为变分参数**，就像波函数一样。

### 3.4 总能量的泛函分解

由 HK 定理，基态总能量可以写为：

```
E[ρ] = T_e[ρ] + V_ext[ρ] + U_ee[ρ]
     = F_HK[ρ] + V_ext[ρ]
```

其中 `F_HK[ρ] = T_e[ρ] + U_ee[ρ]` 是**普适泛函**（与体系无关，只依赖 `ρ`）。

**外势项是唯一已知精确形式的项**：

```
V_ext[ρ] = ∫ ρ(r) v̂_ext(r) dr
```

**关键困难**：`T_e[ρ]` 和 `U_ee[ρ]` 的精确泛函形式**未知**！

> **与 TF 理论的本质区别**：  
> TF 用近似的单粒子密度 `ρ(r) = Σ|φᵢ(r)|²` 和近似动能。  
> DFT 用精确的多体密度 `ρ(r) = N ∫...∫ |Ψ(r, r₂,...,rN)|² dr₂...drN`。  
> HK 定理告诉我们精确泛函**存在**，但没有告诉我们**如何构造**。

## 4. Kohn-Sham 方程：让 DFT 可计算

> **提出者**：W. Kohn & L. J. Sham，1965 年  
> **天才之处**：将无法直接计算的 `F_HK[ρ]` 重新分拆，使大部分可以精确计算

### 4.1 问题所在

HK 定理虽然证明了泛函存在，但动能泛函 `T_e[ρ]` 和电子相互作用 `U_ee[ρ]` 的精确形式未知。TF 的动能近似太差，无法用于原子和分子。

**Kohn-Sham 的解决方案**：引入一个假想的**非相互作用**电子体系，使其基态密度与真实体系相同，通过求解单粒子方程获得动能的精确估计。

### 4.2 KS 假设

**假设**：存在一个非相互作用的参考电子体系，其基态电子密度 `ρ(r)` 与真实相互作用体系完全相同。

对非相互作用体系，波函数是 Slater 行列式，密度可以简洁写为：

```
ρ(r) = Σᵢ |φᵢ(r)|²
```

非相互作用动能可以精确计算：

```
T₀[ρ] = Σᵢ ⟨φᵢ| (-1/2)∇² |φᵢ⟩
```

### 4.3 能量分拆：交换关联能的定义

将总能量重写为：

```
E[ρ] = T_e[ρ] + V_ext[ρ] + U_ee[ρ]
     = T₀[ρ] + V_ext[ρ] + E_cl[ρ] + E_xc[ρ]
```

其中：

- `T₀[ρ]`：非相互作用动能（**可精确计算**）
- `V_ext[ρ] = ∫ ρ(r) v̂_ext(r) dr`：外势能（**精确已知**）
- `E_cl[ρ] = (1/2)∬ ρ(r')ρ(r)/|r'-r| dr'dr`：经典库仑（Hartree）能（**精确已知**）
- `E_xc[ρ]`：**交换关联能**（**未知，需要近似**）

**交换关联能的定义**（它包含了所有被其他项遗漏的物理）：

```
E_xc[ρ] = [T_e[ρ] - T₀[ρ]] + [U_ee[ρ] - E_cl[ρ]]
           ←─ 动能修正 ───→   ←─── 交换+关联 ────→
```

具体包含：
1. **电子交换能**（Pauli 不相容原理引起）
2. **电子关联能**（非相互作用电子需要关联运动的修正）
3. **动能修正**（`T_e` 与 `T₀` 的差）
4. **自相互作用修正**（经典库仑 `E_cl` 引入的自相互作用需要消除）

### 4.4 Kohn-Sham 方程的推导

对总能量泛函关于 `ρ(r)` 做变分（约束条件：`∫ρ dr = N`），引入化学势 `μ` 作为 Lagrange 乘子：

```
μ = δE[ρ(r)] / δρ(r) = δT₀[ρ(r)]/δρ(r) + V̂_eff(r)
```

定义**有效势**：

```
V̂_eff(r) = v̂_ext(r) + Û_cl(r) + V̂_xc(r)

其中：
  Û_cl(r) = ∫ ρ(r')/|r'-r| dr'   (Hartree 势)
  V̂_xc(r) = δE_xc[ρ(r)] / δρ(r)  (交换关联势，由近似给出)
```

这等价于求解一组**单粒子 Schrödinger 方程（Kohn-Sham 方程）**：

```
[ -1/2 ∇ᵢ² + V̂_eff(r) ] φᵢ^KS(r) = εᵢ φᵢ^KS(r)
```

- `φᵢ^KS(r)`：Kohn-Sham 轨道
- `εᵢ`：KS 本征值（注意：**原则上没有严格的单粒子能量物理意义**）
- 密度由 KS 轨道重新构造：`ρ(r) = Σᵢ |φᵢ^KS(r)|²`

**注意**：`V̂_eff(r)` 是非局域的，即它本身依赖 `ρ(r)`，而 `ρ(r)` 又依赖 KS 轨道，形成**自洽循环**。

### 4.5 自洽场（SCF）求解流程

```
初始猜测密度 ρ₀(r)
       ↓
构造有效势 V̂_eff(r) = v̂_ext + Û_cl[ρ] + V̂_xc[ρ]
       ↓
求解 KS 方程，得到 φᵢ^KS(r) 和 εᵢ
       ↓
构造新密度 ρ_new(r) = Σᵢ |φᵢ^KS(r)|²
       ↓
混合密度（防止不收敛）：ρ_in = ω·ρ_new + (1-ω)·ρ_old
       ↓
判断收敛（ρ_in ≈ ρ_old 或能量差 < 阈值）
       ↓ 否
    返回上一步
       ↓ 是
  自洽完成，计算总能量和各物理量
```

**关键物理图像**：多电子问题被化为单电子在**平均场** `V̂_eff(r)` 中运动的问题，所有多体效应被"塞进"了 `E_xc[ρ]`（通常 `< 5%` 的总能量）。

### 4.6 DFT 的两个重要声明

| 声明 | 说明 |
|------|------|
| **DFT 是精确的** | HK 定理是严格数学证明，无任何近似 |
| **精确 DFT 无法实际使用** | `T_e[ρ]` 和 `U_ee[ρ]` 的精确密度泛函形式未知；只有 `V_ext[ρ]` 的密度泛函精确已知 |

> 实际使用的 DFT（Kohn-Sham DFT）引入了两处近似：  
> 1. **KS 假设**：非相互作用体系与真实体系有相同密度（已被大量实践证实是极好的近似）  
> 2. **Exc 近似**：对交换关联能使用 LDA、GGA 等近似

## 5. 交换关联能 Exc：DFT 的核心近似

### 5.1 为什么 Exc 难以精确计算？

`E_xc[ρ]` 包含：
- **交换能**：Pauli 不相容原理，同自旋电子相互回避
- **关联能**：不同自旋电子也因库仑排斥而关联运动
- **动能修正**：真实相互作用体系的动能与非相互作用体系的差值
- **自相互作用修正**：抵消经典库仑项中电子与自身的作用

这些效应高度纠缠，目前**没有已知的精确密度泛函表达式**。

### 5.2 能量量级对比

以 Mn 原子价电子为例（说明为何近似尚可接受）：

```
能量量级：
  库仑能 E_CV, E_VV  >> 动能 T₀ >> 交换能 E_x >> 关联能 E_c

关系：E_c ≈ 0.1 × E_x

LDA 的误差：
  交换能 E_x 被低估约 15%
  关联能 E_c 被高估约 100%

但由于 Exc 只占总能量的一小部分（< 5%），
即使近似误差很大，对总能量的影响仍然可以接受。
```

## 6. 局域密度近似 (LDA)

> **Local Density Approximation**：KS 方法最早的实现方案

### 6.1 基本思路

**假设**：在空间每一个微小区域内，电子密度是均匀的（可用均匀电子气描述）。

对均匀电子气，交换关联能密度 `εxc(ρ)` 可以精确计算（来自 Monte Carlo 模拟等）。

### 6.2 LDA 泛函形式

```
E_xc^LDA[ρ] = ∫ ρ(r) εxc[ρ(r)] dr = ∫ ρ(r)(εx + εc) dr
```

交换关联势：

```
V̂_xc^LDA(r) = δE_xc^LDA / δρ(r) = εxc + ρ · ∂εxc/∂ρ
```

### 6.3 LDA 的特点

**为什么 LDA 能工作**（尽管假设看似粗糙）：

- 只有一小部分总能量被近似
- 误差存在一定的系统性抵消（交换能低估 vs 关联能高估）
- 对慢变密度体系（金属、半导体）效果很好

**已知问题**：

- 过度高估键能（overbinding）：键长偏短
- 氢键太弱
- 强关联体系（过渡金属氧化物）严重失效
- 带隙系统性低估

**LDA 常见泛函**：
- 交换：Slater exchange
- 关联：Vosko-Wilk-Nusair (VWN)

## 7. 广义梯度近似 (GGA)

> **Generalized Gradient Approximation**：LDA 的自然改进

### 7.1 改进思路

真实体系的电子密度是空间非均匀的。GGA 在 LDA 的基础上，将密度**梯度** `∇ρ(r)` 也纳入 `εxc`：

```
E_xc^GGA[ρ] = ∫ f(ρ(r), ∇ρ(r)) dr
```

不同 GGA 方案的区别在于函数 `f(ρ, ∇ρ)` 的选取。

### 7.2 两大流派

| 流派 | 代表人物 | 哲学 |
|------|---------|------|
| **经验派** | Becke | "一切都是合法的"，参数由拟合分子数据库确定 |
| **物理约束派** | Perdew | 泛函必须满足物理规律（标度关系、渐近行为等） |

> 不同 LDA 方案差别很小；不同 GGA 方案可能给出完全不同的结果，使用时须谨慎。

### 7.3 常用 GGA 泛函

| 领域 | 常用泛函 | 特点 |
|------|---------|------|
| 物理（固体） | **PBE** (Perdew-Burke-Ernzerhof) | 基于物理约束，广泛应用于固体计算 |
| 化学（分子） | **BLYP** (Becke + Lee-Yang-Parr) | 经验拟合，对分子键能描述好 |
| 化学（计算） | **B3LYP** (混合) | 计算化学的主力泛函（含 HF 交换） |

**GGA 改进**：对大多数化学键类型（共价、离子、金属、氢键）给出可靠结果。

### 7.4 GGA 常用泛函列表

```
交换泛函：B88 (Becke, 1988)、PW91 (Perdew & Wang, 1991)、PBE
关联泛函：P86 (Perdew, 1986)、LYP (Lee et al, 1988)、PW91、PBE
```

## 8. 泛函层级：Jacob's Ladder

Perdew 提出的泛函精度层级，从"地球"（Hartree 理论）攀向"化学精度的天堂"：

```
Heaven of Chemical Accuracy
         ?           ← 第5层（+非占据轨道信息）
      Hybrid         ← 第4层（+显式占据轨道，如 HF 交换）
       mGGA          ← 第3层（+动能密度 τ，隐式轨道信息）
       GGA           ← 第2层（+密度梯度 ∇ρ）
       LDA           ← 第1层（局域密度）
Earth: Hartree Theory
```

### 8.1 各层级代表泛函

| 层级 | 类型 | 典型泛函 |
|------|------|---------|
| 1 | LDA | Slater + VWN |
| 2 | GGA | PBE (物理), BLYP (化学) |
| 3 | Meta-GGA | TPSS, SCAN, rSCAN |
| 4 | Hybrid GGA | B3LYP, PBE0 |
| 4 | Hybrid meta-GGA | TPSSh, BMK |
| 5 | （开发中） | — |

### 8.2 杂化泛函（Hybrid Functionals）

将一部分 Hartree-Fock 精确交换混入 DFT 交换泛函：

**B3LYP（计算化学主力）**：

```
E_xc^B3 = E_xc^LDA + a(E_x^exact - E_x^LDA) + b·ΔE_x^GGA + c·ΔE_c^GGA
a = 0.20,  b = 0.72,  c = 0.81   (Becke, 1993)
```

**PBE0**：

```
E_xc^0 = E_xc^DFT + a(E_x^exact - E_x^DFT)
a = 0.25   (Perdew, 1998)
```

## 9. DFT 的优势、局限与应用

### 9.1 DFT 相比 Hartree-Fock 的优势

| 方面 | Hartree-Fock | DFT (KS) |
|------|-------------|---------|
| 计算量 | O(N⁴) | O(N³) |
| 交换能 | 精确处理 | 近似处理 |
| 关联能 | **完全忽略** | 近似纳入 |
| 精度 | 较差（缺关联） | 通常更好 |
| 适用体系 | 分子、小体系 | 分子 + 固体 + 大体系 |

> KS 方程的计算量与 Hartree 方程同级，但物理内容深刻得多——有效势原则上包含了所有相互作用效应。

### 9.2 DFT 的已知局限

| 问题 | 原因 |
|------|------|
| **强关联体系**（过渡金属氧化物、硫化物、锕系元素） | 关联能主导，LDA/GGA 失效 |
| **Van der Waals 相互作用** | 长程色散力，密度梯度近似无法捕捉 |
| **激发态** | HK 定理只针对基态（时变 DFT TD-DFT 可处理部分） |
| **带隙低估** | 系统性低估半导体/绝缘体带隙（自相互作用误差） |
| **开壳层体系** | 需要自旋极化 DFT |

### 9.3 DFT 的成功应用领域

**表面化学与异相催化**：
- 吸附结构与吸附能
- 金属、陶瓷、半导体、分子筛
- 反应路径与过渡态

**材料科学**：
- 体相、表面、界面、缺陷

## 10. 核心公式速查表

> **注意**：GitHub 原生 Markdown 不渲染 LaTeX，以下公式用代码块格式呈现。

### Thomas-Fermi

```
T_TF = C_F ∫ ρ^(5/3)(r) dr         (C_F ≈ 2.871)

⟨H⟩_TF = C_F ∫ ρ^(5/3)(r₁) dr₁ 
         + ∫ v(r₁)ρ(r₁) dr₁ 
         + (1/2)∬ ρ(r₁)ρ(r₂)/|r₁-r₂| dr₁dr₂
```

### HK 定理与总能量

```
E[ρ] = T_e[ρ] + V_ext[ρ] + U_ee[ρ]  (精确形式，泛函未知)
V_ext[ρ] = ∫ ρ(r) v̂_ext(r) dr      (唯一精确已知的项)
```

### Kohn-Sham 分拆

```
E[ρ] = T₀[ρ] + V_ext[ρ] + E_cl[ρ] + E_xc[ρ]

T₀[ρ] = (1/2) Σᵢ ⟨φᵢ^KS | ∇ᵢ² | φᵢ^KS⟩    (非相互作用动能，精确)

E_cl[ρ] = (1/2)∬ ρ(r')ρ(r)/|r'-r| dr'dr   (经典库仑能，精确)

E_xc[ρ] = [T_e - T₀] + [U_ee - E_cl]       (交换关联能，需近似)
```

### KS 方程（有效势）

```
[-1/2 ∇ᵢ² + V̂_eff(r)] φᵢ^KS(r) = εᵢ φᵢ^KS(r)

V̂_eff(r) = v̂_ext(r) + Û_cl(r) + V̂_xc(r)

Û_cl(r) = ∫ ρ(r')/|r'-r| dr'

V̂_xc(r) = δE_xc[ρ(r)] / δρ(r)

ρ(r) = Σᵢ |φᵢ^KS(r)|²
```

### LDA

```
E_xc^LDA[ρ] = ∫ ρ(r) εxc[ρ(r)] dr

V̂_xc^LDA(r) = εxc + ρ · ∂εxc/∂ρ
```

### GGA

```
E_xc^GGA[ρ] = ∫ f(ρ(r), ∇ρ(r)) dr
```
